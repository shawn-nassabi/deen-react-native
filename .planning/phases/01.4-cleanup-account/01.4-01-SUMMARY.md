## Decisions Made

- `Linking.openURL` (from react-native) replaces `openBrowserAsync` — zero new dependencies, built-in
- `signOut()` opts parameter removed — settings.tsx no longer needs it; Supabase has no global signout equivalent anyway
- Delete Account error Alert keeps user on Settings — intentional so they can retry if network fails
- `ComponentProps` must be imported from `'react'`, not `'react-native'` (RN doesn't export it)
