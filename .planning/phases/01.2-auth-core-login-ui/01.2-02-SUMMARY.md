## Decisions Made
- No `Redirect` component in `login.tsx` — per plan rule #1, `_layout.tsx` handles auth routing. Including Redirect causes flicker when the component mounts.
- Error message fallback always shows generic "Something went wrong" copy — explicit only for known Supabase error string "Invalid login credentials". This avoids leaking internal error messages.
- `Don{"'"}t` JSX entity workaround used in link text to satisfy `react/no-unescaped-entities` lint rule (consistent with plan template).
